Project: Enterprise CI/CD Pipeline

This project demonstrates a production-grade automated deployment pipeline using a distributed Jenkins architecture on AWS.

The Architecture

  • Orchestration: Jenkins Master triggers builds but remains lightweight to ensure high availability.
  • Security: Credentials (GitHub PAT) are never stored in Jenkins. The Worker EC2 fetches them securely from AWS Secrets Manager using IAM roles.
  • Execution: The AWS SSM agent allows the Master to securely communicate with Workers without exposing SSH ports.
  • Containerization: The Worker node handles the git clone, docker build, and pushes the image to Amazon ECR.
  • Deployment: Automatic updates to Amazon ECS (Task Definitions & Service) ensure zero-downtime deployments.
